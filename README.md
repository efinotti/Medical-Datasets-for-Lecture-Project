# 📊 Grupo de Estudo em Computação Aplicada — 2026/1

Repositório desenvolvido como parte do **Grupo de Estudo em Computação Aplicada**, com foco em **Fundamentos de Ciência de Dados, Análise Multivariada e Preparação para IA**.

O objetivo do projeto é desenvolver habilidades em **manipulação de dados reais, análise estatística, técnicas multivariadas e interpretação de resultados**, utilizando Python e ferramentas de ciência de dados.

---

# 🎯 Objetivos

Ao longo do semestre, o projeto busca:

- Manipular e organizar datasets reais em Python  
- Aplicar estatística descritiva e análise exploratória de dados (EDA)  
- Trabalhar com dados como vetores e matrizes multivariadas  
- Aplicar redução de dimensionalidade (PCA)  
- Aplicar técnicas de clustering  
- Interpretar resultados de forma técnica e crítica  
- Documentar análises em notebooks científicos  

Essas atividades formam a base para aplicações em:

- Inteligência Artificial  
- Machine Learning  
- Business Intelligence  
- Sistemas de Apoio à Decisão  

---

# 🧠 Estrutura do Projeto

O semestre é dividido em **4 projetos principais**:

## Projeto 1 — Análise Exploratória de Dados (EDA)

- Limpeza do dataset  
- Estatísticas descritivas  
- Visualização de dados  
- Primeiras hipóteses  

## Projeto 2 — Análise Multivariada Exploratória (Ainda não feito)

- Correlação e covariância  
- Visualização multivariada  
- Interpretação de relações entre variáveis  

## Projeto 3 — Redução de Dimensionalidade (PCA) (Ainda não feito)

- Padronização dos dados  
- PCA com 2 e 3 componentes  
- Variância explicada  
- Interpretação dos componentes  

## Projeto 4 — Clustering e Ponte com IA (Ainda não feito)

- K-means (ou algoritmo similar)  
- Avaliação de clusters  
- Visualização em espaço reduzido  
- Conexão com problemas de IA

# 📊 Dataset

Inicialmente, o grupo trabalhou com um dataset multivariado da área da saúde utilizado em ensino e pesquisa.

A partir das próximas semanas, será utilizado um **dataset inédito sobre vacinação da COVID-19**, que será tratado ao longo dos próximos projetos.

- Agrupamento de padrões de vacinação  
- Interpretação multivariada dos dados  

Esse novo dataset será utilizado para aprofundar a aplicação prática das técnicas estudadas no grupo.
